# Dashboard: Control de Variabilidad Operacional Mina–Planta

## Modelo de Analítica para Reducción de Variabilidad en Operaciones Mineras

Este dashboard permite monitorear la variabilidad entre Mina y Planta, identificando desviaciones, eventos anómalos y su impacto económico.

## KPIs implementados

- Toneladas plan vs real acumuladas
- % cumplimiento del plan
- Desviación estándar de producción real
- Impacto económico estimado

## Visualizaciones

- Línea: Plan vs Real
- Análisis de variabilidad
- Tabla de outliers
- Mapa conceptual de cuellos de botella

## Filtros

- Rango de fechas
- Turno
- Frente mina
- Línea planta
- Equipo

## Cómo ejecutarlo

1. Descargar la carpeta.
2. Abrir `index.html` en el navegador.
3. No requiere servidor.

## Nota

Los datos son simulados y utilizados únicamente con fines demostrativos.
